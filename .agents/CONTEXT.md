# Project Context: Ma3 Routes

Ma3 Routes (Matatu Route Helper) is a specialized navigation assistant designed for Nairobi's informal transit network. It focuses on providing a high-legibility, minimalist interface optimized for fast-paced, real-world commuting.

## Project Documentation

For detailed information on specific areas of the project, refer to the following documents located in the `docs/` directory:

- **[Architecture](../docs/ARCHITECTURE.md)**: Overview of the application structure, technical stack, and system design.
- **[Design System](../docs/DESIGN.md)**: Specifications for the visual language, including the "Matatu Yellow" high-contrast theme, typography (Inter), and layout principles.
- **[Data Model & Schema](../docs/schema.sql)**: The SQL schema defining routes, stages, corrections, and destinations.
- **[Data Strategy](../docs/DATA.md)**: Information regarding data sourcing, structures, and handling within the application.

## Core Technologies
- **Platform**: Android
- **UI Framework**: Jetpack Compose
- **Design Philosophy**: Utility-first, High-contrast Dark Mode and Light Mode.
- **Language**: Kotlin
